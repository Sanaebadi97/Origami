# Origami

#### This project use the following technologies:

> - Retrofit
> - Room
> - **Hilt DI**
> - Kotlin Coroutines
> - Flow
> - View Model
> - Navigation
> - Repository pattern
> - **Clean Architecture**
> - **JetPack Compose**
